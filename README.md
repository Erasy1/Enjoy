# ENJOY – Movie Recommendation Web Application

## Description
ENJOY is a web application built with Flask that provides movie and series recommendations.

## Features
- User authentication
- Movie and series browsing
- Personalized recommendations
- Onboarding questions
- Watch and settings pages

## Technologies
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite
- TMDB API

## Project Structure
- app.py – main backend application
- templates/ – HTML templates
- static/ – CSS, JS, images
